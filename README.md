Веб-приложение на Django для анализа вакансий и рынка труда в сфере IT.

## Описание

Проект представляет собой Django-приложение для сбора и визуализации данных о вакансиях с сайта HeadHunter (hh.ru). Приложение позволяет анализировать: 
- Востребованность профессий
- Географическое распределение вакансий
- Требуемые навыки
- Свежие вакансии

## Технологии

- **Django 4.1.5** - веб-фреймворк
- **Python 3.x**
- **SQLite** - база данных
- **pandas** - обработка данных
- **openpyxl** - работа с Excel файлами
- **django-jinja** - шаблонизатор

## Установка

1. Клонируйте репозиторий: 
```bash
git clone https://github.com/likip3/djangoProjectUzenkov.git
cd djangoProjectUzenkov
```

2. Установите зависимости:
```bash
pip install django pandas openpyxl requests django-jinja
```

3. Выполните миграции:
```bash
python manage.py migrate
```

4. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

5. Запустите сервер:
```bash
python manage.py runserver
```

## Функционал

### Модели данных
- **Customer** - тестовая модель клиентов
- **DemandData** - данные о востребованности (изображения графиков)
- **GeoData** - географические данные (Excel файлы)
- **SkillsData** - данные о навыках (Excel файлы)

### Основные страницы
- `/` - главная страница
- `/demand/` - востребованность профессий
- `/geo/` - географическое распределение
- `/skills/` - требуемые навыки
- `/recent/` - последние вакансии с hh.ru
